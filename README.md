# Booking Homestay System

## Description
A web-based homestay booking system developed using ASP.NET Core.
The system allows users to browse homestays, add bookings to cart,
and complete online payments via VNPAY.

## Features
- User authentication & authorization using JWT
- Homestay listing and booking management
- Cart and checkout flow
- Online payment integration (VNPAY Sandbox)
- Role-based access control (Admin / User)

## Technologies
- ASP.NET Core MVC & Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- VNPAY Sandbox
- Git & GitHub
